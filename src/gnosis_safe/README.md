# Safe
Safe 是一个广受欢迎的多签钱包解决方案，为个人、企业和去中心化自治组织（DAO）提供了安全且灵活的数字资产管理方式。它通过智能合约实现，支持以太坊及其兼容链，如 Polygon、Binance Smart Chain，以及测试网 Sepolia 等。
